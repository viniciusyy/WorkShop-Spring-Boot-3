# WorkShop Spring Boot 3

WorkShop Spring Boot 3 é um projeto desenvolvido em Java utilizando o framework Spring Boot 3. O objetivo é demonstrar a construção de uma API RESTful com integração ao banco de dados utilizando JPA e Hibernate.

## Funcionalidades

- **API RESTful:** Exposição de endpoints para operações CRUD.
- **Spring Boot 3:** Utilização das melhores práticas e recursos da nova versão do Spring Boot.
- **JPA e Hibernate:** Mapeamento objeto-relacional (ORM) para comunicação com o banco de dados.
- **Banco de Dados:** Suporte para bancos de dados relacionais configuráveis.
- **Spring Boot DevTools:** Ferramentas para facilitar o desenvolvimento.


## Requisitos

- **Java JDK 17 ou superior**: Verifique se o Java está instalado.
- **Maven**: Necessário para gerenciar dependências e construir o projeto.
- **Banco de Dados**: MySQL ou H2 (dependendo da configuração do projeto).
- **IDE Recomendada**: IntelliJ IDEA, Eclipse ou VS Code com suporte para Spring Boot.
